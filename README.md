# Ship & Container Management System

## Overview
This project is a Java application that simulates the management of ships and cargo containers. It demonstrates object-oriented programming concepts such as inheritance, polymorphism, encapsulation, and collections.

## Features
- Create and manage different ships.
- Create different types of cargo containers.
- Assign containers to ships.
- Store and manage ships using Java collections (`ArrayList`).
- Demonstrate object-oriented design and class relationships.

## Technologies
- Java
- Eclipse IDE

## Project Structure
- `src/` – Java source code
- `bin/` – Compiled class files

## How to Run
1. Open the project in Eclipse.
2. Run the `Main.java` class.
3. The application will execute the simulation.
